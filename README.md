# IEEE1451.0_TEDS_Editor
An editor for IEEE 1451.0 TEDS

## Install PyQT5
```python -m pip istall pyqt5-tools```

## Compile qt5.ui file to Python file
```python -m PyQt5.uic.pyuic -x teds_editor.ui -o teds_editor.py```

## Run with
```python main_teds.py```
